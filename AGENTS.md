# AGENTS.md

## 协作方式

- 新线程进入项目时，先读取 `docs/origin.md`，再读取 `README.md`、`docs/03-phase-one-spec.md` 和 `ai/context/project-brief.md`。
- 遇到复杂任务时，主动请示是否释放多 agent 协作。
- 如果开启多 agent 协作，根据分发任务的复杂程度为子 agent 设置较低版本的 AI 模型，例如 gpt4。
- AI 相关的项目内工作方法沉积在 `ai/` 目录中；先更新这些普通 Markdown，再考虑抽成可安装 skill。

## 项目边界

未形 / primordia 只探索本地安全沙盒中的低等数字生命。

禁止方向：

- 真实互联网中的自我传播
- 权限试探、逃逸、提权
- token / 密钥搜寻
- 资源掠夺
- 绕过沙盒或身份边界

允许方向：

- 本地、可观察、可终止的二维信息环境仿真
- 资源、能量、繁殖、变异、死亡等生命机制
- 安全边界、异常隔离和可解释的演化记录
