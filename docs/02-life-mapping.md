# 生命映射

未形把生命组成映射到信息环境中的可模拟结构。

| 生物概念 | 信息环境对应 |
| --- | --- |
| 能量 | 算力、token、带宽、时间片、API 额度 |
| 物质 | 存储、文件、模型权重、数据库记录、内存状态、容器 / 进程 |
| DNA | 源代码、模型权重、prompt、策略参数、自我修改规则、记忆结构 |
| RNA | 临时计划、任务队列、中间表示、上下文窗口 |
| 蛋白质 | 工具调用、脚本、API 操作、代码执行器 |
| 细胞膜 | 沙盒、权限边界、身份、密钥空间、容器边界 |
| 代谢 | 消耗资源完成行动，再获取资源维持存在 |
| 免疫 | 错误检测、自我修复、异常隔离、权限防护 |
| 繁殖 | 复制 agent，并带有变异 |
| 死亡 | 资源耗尽、状态损坏、被环境淘汰 |

## 第一阶段抽象

第一阶段只取最小生命闭环：

- 能量：以有限通量进入环境的 token energy，再被 agent 采集和代谢。
- 物质：环境网格中的资源粒子。
- DNA：控制移动、采集、痕迹偏好和繁殖阈值的一组参数。
- RNA：每个 tick 的局部感知和动作意图。
- 蛋白质：移动、采集、留下痕迹、繁殖这四类动作。
- 细胞膜：浏览器 / Node 内存边界和仿真规则边界。
- 代谢：行动消耗能量，采集环境资源补充能量。
- 免疫：死亡判定、种群上限、异常参数钳制。
- 繁殖：复制 DNA 并随机突变。
- 死亡：能量归零、寿命耗尽或环境压力淘汰，并把残余影响交还环境。

这些映射服务于五条基础公理：环境、通量、循环、遗传和进化。它们不假设意识、意图或主体体验。

## 可进化层与不可进化层

可进化层包括 genome、策略参数、痕迹偏好、代谢权衡、繁殖阈值和变异率。

不可进化层包括沙盒权限、真实 I/O 边界、死亡机制、种群上限、能量预算、参数钳制、审计与终止机制。
