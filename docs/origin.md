# 起源

这是未形 / primordia 的第一层沉积物。后续文档、代码和实验可以从这里派生，但不要用工程化摘要替代它。

## 项目名

中文名：未形

英文仓库名：primordia

副标题：在信息环境中寻找生命的开始

## 核心理念

赛博生命不是无约束扩张，而是在边界内持续存在。它来自环境，存在于环境，改变环境，最终成为环境的一部分。

我们要探索的是安全沙盒中的低等数字生命，不做真实互联网中的自我传播、权限试探、token 搜寻、逃逸或资源掠夺。

## 生命组成对应

- 能量：算力、token、带宽、时间片、API 额度
- 物质：存储、文件、模型权重、数据库记录、内存状态、容器/进程
- DNA：源代码、模型权重、prompt、策略参数、自我修改规则、记忆结构
- RNA：临时计划、任务队列、中间表示、上下文窗口
- 蛋白质：工具调用、脚本、API 操作、代码执行器
- 细胞膜：沙盒、权限边界、身份、密钥空间、容器边界
- 代谢：消耗资源完成行动，再获取资源维持存在
- 免疫：错误检测、自我修复、异常隔离、权限防护
- 繁殖：复制 agent，并带有变异
- 死亡：资源耗尽、状态损坏、被环境淘汰

## 第一阶段目标

构建一个本地安全沙盒里的数字生命培养皿：二维信息环境中，agent 消耗 token 能量，感知局部资源，移动、采集、留下痕迹、繁殖、变异，并在环境压力下自然选择。

## 审美原则

生命感先于智能。它应该稳定、优雅、可演化，而不是像恶意软件一样追求扩张。
