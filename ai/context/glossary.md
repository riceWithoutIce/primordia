# 词汇表

## 未形

项目中文名。指向尚未固定形态的生命起点，也指项目本身在文档、代码和实验中逐渐成形。

## primordia

英文仓库名。含义接近原基、初始形成物。

## 培养皿

本地安全沙盒里的二维信息环境。agent 的全部行为都发生在培养皿内。

## 环境

包含资源、痕迹、压力、边界和观察机制的整体。未形中的生命来自环境，也会改变环境。

## 细胞膜

沙盒、权限、身份和边界。细胞膜不是对生命的否定，而是生命得以稳定存在的结构。

## 生命感

不是智能问答能力，而是持续存在、代谢、适应、繁殖、死亡、改变环境并被环境选择的可观察过程。

## token 能量

第一阶段的模拟能量单位。它可以隐喻时间片、算力或 API 额度，但在代码中只作为本地数值资源存在。
