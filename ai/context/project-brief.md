# 项目简报

未形 / primordia 是一个本地安全沙盒中的数字生命培养皿。

它探索的不是高智能助手，也不是自我传播系统，而是低等数字生命在边界内持续存在的条件：能量、物质、DNA、RNA、蛋白质、细胞膜、代谢、免疫、繁殖和死亡如何在信息环境中形成一个可观察的生命闭环。

基础条件是本地信息生物圈：能量以有限通量进入环境，经资源、痕迹、压力与死亡回收循环，再参与下一代生命的选择。

核心原则：

- 环境归属原则：生命在环境中形成、维持、改变，并最终回到环境。
- 生命驱动原则：生命延续自身 genome，并通过痕迹让相近形态在未来更可能出现。
- 有界进化原则：变异不能越过细胞膜，优势必须伴随生态代价。
- 生存循环原则：生命必须通过感知、代谢、行动、资源获取、繁殖或修复、死亡与回归环境的闭环维持存在。
- 观察者原则：生命过程必须可观察、可暂停、可重置、可回放。
- 非拟人化限制：第一阶段不假设意识、意图或主体体验。

## 当前优先级

1. 保持安全边界清晰。
2. 让培养皿稳定、可观察、可暂停、可重置。
3. 让生命感先于智能感出现。
4. 记录实验参数和观察结果。
5. 完成死亡回收、谱系观察和生态反馈。
6. 再逐步引入更复杂的生态结构。

## 不要做

- 不要实现真实网络传播。
- 不要让 agent 读取真实文件、环境变量、token 或密钥。
- 不要研究逃逸、提权或权限试探。
- 不要为了“更像生命”牺牲可终止性和可解释性。
